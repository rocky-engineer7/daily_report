# わかったこと
- 正規表現
- SELECT * FROM テーブル名 WHERE カラム名 ~ 正規表現

- SQLの`CASE`文は、条件に基づいて異なる値を返すために使用される条件分岐の機能

- LIMIT関数はレコードの上限を設定する
- SELECT * FROM テーブル LIMIT 設定レコード数
- OFFSET は開始位置を設定する - 1  11行目から開始したい場合は11-1と記述する
