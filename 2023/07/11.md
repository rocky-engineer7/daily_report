# わかったこと
- オブジェクトの値の部分には、文字列や数値だけでなく、オブジェクトを用いることが出来る
- オブジェクト名.プロパティ.プロパティで呼び出し
- アロー関数() =>
- const 定数名 = (引数名) => {処理} 引数は,で複数いける
- 関数呼び出し 定数名(値);
- 戻り値　return 値;
