# わかったこと
- push 新しい要素を配列の一番最後に追加
- forEach 配列の中の要素を1つずつ取り出して、全ての要素に繰り返し同じ処理を行うメソッド
- 引数に入っている関数はコールバック関数
- find 条件式に合う1つ目の要素を配列の中から取り出すメソッド
- filter 記述した条件に合う要素のみを取り出して新しい配列を作成する
- map 配列内のすべての要素に処理を行い、その戻り値から新しい配列を作成するメソッド
